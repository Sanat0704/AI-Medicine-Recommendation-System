# 💊 AI Medicine Recommendation System

A modern AI-powered healthcare analytics dashboard built using Python, Streamlit, Pandas, and Machine Learning.

## 🚀 Features

* 🔍 Medicine Search
* 🩺 Disease-Based Medicine Search
* ⚠ Side Effects Analysis
* 📊 Healthcare Analytics Dashboard
* 📈 Therapeutic Class Visualization
* 🌙 Modern Netflix-Style UI
* 🤖 AI-Powered Healthcare Insights

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas
* Machine Learning
* Data Visualization

---

## 📂 Dataset

Healthcare medicine dataset containing:

* 248k+ medicine records
* therapeutic classes
* side effects
* disease uses
* habit-forming analysis
